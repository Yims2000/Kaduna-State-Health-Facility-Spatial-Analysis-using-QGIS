# Kaduna-State-Health-Facility-Spatial-Analysis-using-QGIS
This project presents a spatial analysis of Primary Healthcare Centres (PHCs) across Kaduna State, Nigeria, using QGIS. The analysis evaluates the geographical distribution of healthcare facilities through GIS techniques to support healthcare planning, service accessibility assessment, and evidence-based decision-making.

## Data Source

Kaduna State Bureau of Statistics (KDBS)

The datasets used in this project were obtained from the Kaduna State Bureau of Statistics for educational and analytical purposes. The dataset is a sensitive one

---

## Objectives

- Map Primary Healthcare Centre locations across Kaduna State.
- Examine the spatial distribution of health facilities.
- Generate service coverage buffers around PHCs.
- Support healthcare accessibility assessment.
- Produce publication-quality thematic maps.
